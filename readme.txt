1. Download the Repo.
2. Set config.ini to your Serverdetails. For localhost type 127.0.0.1. Change the Betakeyneeded to 0 if no BetaKey is needed. 
   Change it to 1 if a BetaKey is needed.
3. Change Servername and Realmlist in index.php
4. Place files into your Website Folder
5. Import the sql file into your authdb, if you activate the BetaKeyoption.
6. Site is ready.
